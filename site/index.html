From install to recalculation in eight lines.

No DOM, no canvas, no worker setup. The engine is a plain class: feed it cells, read values, subscribe to what changed.

quickstart.ts
import { Engine } from "gridcalc";

const engine = Engine.buildEmpty();
engine.setCellContents({ sheet: 0, col: 0, row: 0 }, 1200);
engine.setCellContents({ sheet: 0, col: 0, row: 1 }, 800);
engine.setCellContents({ sheet: 0, col: 1, row: 0 }, "=SUM(A1:A2)*1.21");

engine.getCellValue({ sheet: 0, col: 1, row: 0 }); // 2420
